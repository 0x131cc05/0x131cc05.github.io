AGC035E Develop

发表于 2020-07-09 更新于 2020-07-11 分类于 题解

题意

黑板上写着$-10^{18}\sim 10^{18}$间的所有整数，每一个整数都出现了一次。你可以进行任意多次如下操作：

  • 选择一个$1$到$n$之间，并且出现在黑板上的整数$x$，擦去这个整数。
  • 如果$x-2$不在黑板上，写下$x-2$。
  • 如果$x+k$不在黑板上，写下$x+k$。

问最终黑板上剩下的整数集合可能有多少种。
